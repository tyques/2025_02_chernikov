# Генератор таблицы умножения

Простое консольное приложение, которое генерирует и выводит на экран таблицу умножения заданного размера.

## Сборка проекта

Чтобы собрать исполняемый JAR-файл, выполните следующую команду из корневого каталога проекта:

**Для Windows:**
```bash
.\gradlew.bat build
```

**Для macOS/Linux:**
```bash
./gradlew build
```

Эта команда скомпилирует код и создаст JAR-файл в директории `build/libs/`.

## Запуск приложения

После успешной сборки вы можете запустить приложение, используя следующую команду:
```bash
java -jar task1\build\libs\task1-1.0-SNAPSHOT.jar
```
Программа предложит вам ввести в консоли размер таблицы (целое число от 1 до 32).
